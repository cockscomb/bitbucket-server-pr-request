# bitbucket-server-pr-request

[git-pr-release](https://github.com/motemen/git-pr-release) for BitBucket Server

## Usage

### Arguments

```
--bitbucket-server URL
    BitBucket Server URL
--username string
    BitBucket Server username
--password string
    BitBucket Server password
--project string
    Project name
--repository string
    Repository name
--from branch name
    from branch name
--to branch name
    to branch name
```
